On-Line Archiving Facility

OLAF is a web site that allows individuals to submit their data to the Planetary Data System so that it can be archived and made available to the public. Submitted data is reviewed for quality and completeness and given a citation when accepted. This software is made available by the Small Bodies Node (SBN) of the PDS.

PDS3 OLAF

https://sbnapps.psi.edu/olaf-pds3

This is the OLAF submission site for PDS3 data products.

Supported Data Types:

  • FITS Images
  • Fixed-Width Tables
    • Time Series/Lightcurves
    • Telescope Spectral
    • Laboratory Spectral
    • Multipart Fixed-Width Tables
  • General Tables
  • Documents

PDS4 OLAF

https://sbnapps.psi.edu/olaf

The original (or "classic") update to OLAF. This version of OLAF makes additional PDS4 data types available to data providers.

Supported Data Types:

  • FITS Images
  • Delimited Tables
  • Fixed-Width Tables
  • Complex Tables
  • Documents

PDS4 OLAF

* Public Beta

https://sbnapps.psi.edu/olaf-node

The latest public beta version of the PDS4 OLAF submission site. While still under development, the application is available to beta users for submitting certain data types to the PDS.

Supported Data Types:

  • Delimited (CSV) Tables
  • FITS Images
  • Documents
