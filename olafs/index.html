On-Line Archiving Facility

OLAF is a web application that allows individuals to submit their data to the Planetary Data System so that it can be archived and made available to the public. Submitted data is reviewed for quality and completeness and given a citation when accepted. This software is made available by the Small Bodies Node (SBN) of the PDS.

If you need access to a previous version of OLAF, please contact us at olaf@psi.edu

Open

Currently Supported Data Types:

  • Delimited (CSV) Tables
  • Fixed-Width Tables
  • Multi-Part Tables
  • FITS Images (Extension support coming soon)
  • 2D Array Images
  • Documents

More data types in development. If you need something not listed here, let us know at olaf@psi.edu
